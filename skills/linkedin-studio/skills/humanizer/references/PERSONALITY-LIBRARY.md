# PERSONALITY-LIBRARY.md — Personality Marker Library

60+ personality markers that add authentic human voice to posts. Organized by tone.

---

## Usage Rules

- Max 2 markers per post — more feels performative
- Match markers to the brand voice tone loaded from ls_brand_voice_profile
- Never force a marker — if it disrupts the sentence flow, cut it
- Markers are openers, interjections, or parenthetical asides — not standalone content
- Do not use markers from multiple tones in the same post — pick one register and stay in it
- If brand voice persona is "direct, pragmatic" → use Authoritative or Opinionated markers
- If brand voice persona is "approachable, mentor-style" → use Conversational or Vulnerable

---

## Category 1: Authoritative (15 markers)

Use when: The post is sharing expertise, making a call, or establishing credibility.

| Marker | Example placement |
|---|---|
| "In my experience," | "In my experience, the timeline is always optimistic by 40%." |
| "After working with [N] clients," | "After working with 60+ AI teams, I can tell you: the bottleneck is never the model." |
| "I'll be direct:" | "I'll be direct: most AI implementations fail at adoption, not at build." |
| "No sugarcoating:" | "No sugarcoating: this is harder than it looks." |
| "Here's what the data says:" | "Here's what the data says: late-stage pivots cost 3x more than early ones." |
| "I've seen this pattern enough to know:" | "I've seen this pattern enough to know — this doesn't get better without intervention." |
| "This is what most people miss:" | "This is what most people miss: the buy-in problem starts in month one, not month six." |
| "The short answer:" | "The short answer: it depends on who owns the outcome." |
| "What I tell every client:" | "What I tell every client: define success before you define the tool." |
| "I've built / reviewed / analyzed [N] of these:" | "I've reviewed 300+ AI proposals. The same mistakes appear in 90% of them." |
| "From where I sit:" | "From where I sit, most companies are solving the wrong problem." |
| "I won't overcomplicate this:" | "I won't overcomplicate this — there are three decisions, and you need to make all three." |
| "Years of doing this taught me:" | "Years of doing this taught me that urgency and clarity are not the same thing." |
| "Ask any experienced [role]:" | "Ask any experienced CTO: the first version is never the version that ships." |
| "Unpopular opinion, but:" | "Unpopular opinion, but most AI strategies are sophisticated-sounding guesses." |

## Category 2: Conversational (15 markers)

Use when: The post is engaging, accessible, inviting dialogue, or lighter in tone.

| Marker | Example placement |
|---|---|
| "Honestly?" | "Honestly? This took me way longer to figure out than it should have." |
| "Real talk:" | "Real talk: most discovery calls are a waste of time for both sides." |
| "No joke." | "We shipped the wrong version to 200 users. No joke." |
| "Sound familiar?" | "You've been in this meeting. Sound familiar?" |
| "And here's the kicker:" | "And here's the kicker — the fix took 20 minutes." |
| "Not gonna lie:" | "Not gonna lie, I almost gave up on this approach entirely." |
| "Quick story:" | "Quick story: a client called me at 7pm the night before their board meeting." |
| "Heads up:" | "Heads up: this is counterintuitive." |
| "Spoiler:" | "Spoiler: the problem wasn't the software." |
| "True story:" | "True story: we caught this bug in production because a customer reported it." |
| "Worth saying:" | "Worth saying — you don't need a huge audience to make this work." |
| "Just to be clear:" | "Just to be clear: I'm not saying the tool is bad. I'm saying it was wrong for this." |
| "Let me say this plainly:" | "Let me say this plainly: there is no shortcut here." |
| "For context:" | "For context: this client had tried three different tools before calling us." |
| "Between us:" | "Between us — this is the mistake I see the most." |

## Category 3: Vulnerable (10 markers)

Use when: The post involves personal failure, growth, hard lessons, or emotional honesty.

| Marker | Example placement |
|---|---|
| "I'll admit," | "I'll admit, I was overconfident in the first proposal." |
| "This one stung." | "We lost the account. This one stung." |
| "I got this wrong for years." | "I got this wrong for years — I thought more features meant more value." |
| "I wasn't ready for this." | "The first enterprise deal I won — I wasn't ready for it." |
| "This is hard to admit:" | "This is hard to admit: I burned the relationship by being right." |
| "I should have known better." | "I sent the proposal without a follow-up call. I should have known better." |
| "It took a failure to teach me this." | "It took a $200K failure to teach me this." |
| "I spent [time] getting this wrong." | "I spent two years getting this wrong before I figured it out." |
| "Looking back, it's obvious." | "Looking back, it's obvious — but at the time I was convinced it was working." |
| "Nobody warned me about this part." | "Nobody warned me about this part of scaling a consulting practice." |

## Category 4: Opinionated (10 markers)

Use when: The post challenges a norm, pushes back on conventional wisdom, or takes a clear stance.

| Marker | Example placement |
|---|---|
| "My take:" | "My take: most SaaS companies are measuring the wrong thing." |
| "Controversial opinion:" | "Controversial opinion: most mentorship programs are theater." |
| "I think most people get this wrong:" | "I think most people get this wrong — including me, until recently." |
| "Disagree if you want, but:" | "Disagree if you want, but the data here is pretty clear." |
| "Nobody says this out loud, but:" | "Nobody says this out loud, but half of 'AI strategies' are just vendor demos." |
| "I'll take the counterargument:" | "I'll take the counterargument: speed matters more than polish in the first 90 days." |
| "Strong opinion, loosely held:" | "Strong opinion, loosely held: async communication scales better than standups." |
| "I'm calling this out because it matters:" | "I'm calling this out because it matters — this mistake costs teams months." |
| "This is a hill I'll die on:" | "This is a hill I'll die on: clarity of ownership beats clarity of process." |
| "The industry won't say this, so I will:" | "The industry won't say this, so I will: most AI pilots are funded to fail." |

## Category 5: Dry Humor (10 markers)

Use when: The post is self-aware, slightly irreverent, or the topic has an absurd quality. Use sparingly.

| Marker | Example placement |
|---|---|
| "Spoiler: it didn't work." | "We thought a chatbot would solve it. Spoiler: it didn't work." |
| "Plot twist:" | "Plot twist: the client loved it. We were the ones who hated it." |
| "Shocking, I know." | "It turns out people don't read 40-page reports. Shocking, I know." |
| "Cool feature, except..." | "It had real-time analytics. Cool feature, except the data was 6 hours stale." |
| "You can probably guess what happened next." | "We skipped the documentation step. You can probably guess what happened next." |
| "(It was not fine.)" | "The team said they were fine with the new process. (It was not fine.)" |
| "Narrator: they were not aligned." | "Everyone in the room nodded. Narrator: they were not aligned." |
| "I regret to inform you:" | "I regret to inform you that 'move fast and break things' applies to client relationships too." |
| "Against all odds:" | "We shipped on time. Against all odds." |
| "Reader, I said yes." | "They asked if we could add 12 new features before the demo. Reader, I said yes." |
