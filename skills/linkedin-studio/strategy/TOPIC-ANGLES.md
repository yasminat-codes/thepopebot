## Using This File

**Research-engine:** Load these angles as seed topics alongside live Google Trends, Reddit, and LinkedIn creator data. Cross-reference against topic_bank to avoid duplicating recent posts.

**Content-writer:** When generating a post, check if the assigned niche has relevant angles here. Use as inspiration, not verbatim copy.

**Content-calendar:** When planning a week, verify niche rotation. If the last 2 weeks featured e-commerce angles, pick a different niche this week.
